## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

The app will be available at `https://yourusername.github.io/cut/`

## Configuration

### Google AdSense

Replace `ca-pub-XXXXXXXXXXXXXXXX` in `index.html` and `src/components/AdBanner.tsx` with your actual AdSense publisher ID.

### Ad Slots

Update the ad slot IDs in `src/App.tsx` with your actual ad unit IDs.
